## 배경

### 1차 마케팅 결과
- **쓰레드, X, 에브리타임** 모두 게시 완료
- 전 채널 조회수 매우 낮음 → 유의미한 유입/검증 불가
- **디스콰이엇, 긱뉴스** → 기술 중심이라 후순위로 보류
- 오가닉 채널만으로는 초기 트래픽 확보가 어려운 상황

- 에브리타임은 홍보게시판 특성상 노출 자체가 제한적
- **직접 도달 가능한 커뮤니티**에 올리는 게 더 효과적

## Phase 1: 카카오 오픈채팅방 (1인 개발자 700명)

### 왜 이 채널인가?
- **700명이 이미 모여 있음** — 알고리즘 의존 X, 직접 도달
- 1인 개발자 = 사이드프로젝트에 공감 + 피드백 줄 의향 높음
- 동료 메이커의 프로덕트를 써보고 응원하는 문화
- 채팅방이라 실시간 반응 확인 가능

### 글 작성 방향
- **절대 하지 말 것:** 서비스 링크만 던지기, 광고체, 장문
- **핵심 톤:** "만들었는데 써보시고 피드백 주세요" — 메이커 동료로서의 요청
- **구조:**
  1. 한 줄 소개 (뭘 만들었는지)
  2. 왜 만들었는지 (개인적 동기 1~2문장)
  3. 핵심 기능 3가지 (짧게)
  4. 링크 + 피드백 요청
- **길이:** 10줄 이내 — 채팅방이므로 짧을수록 좋음

### 홍보글 예시
```

• 같은 사건의 여러 언론사 원문 비교
```

### 기대 효과
- 700명 중 5~10% 클릭 가정 → **35~70 UV**
- 메이커 특성상 피드백 가능성 높음 (Google Form 또는 직접 채팅)
- 긍정적이면 2차 공유(본인 SNS) 가능성 있음

### 주의사항
- 채팅방 규칙 확인 (홍보 허용 여부, 빈도 제한)
- 한 번만 올리고, 반응 보기 — 반복 홍보는 역효과
- 질문이 오면 성실하게 답변 → 신뢰 구축

### 추천: Meta 광고 (₩5,000/일 × 7일 = ₩35,000)

**이유:**
- 최소 예산이 가장 낮음 (₩5,000/일부터 가능)
- 20~30대 + 뉴스/시사 관심사 타겟팅 가능
- Threads/Instagram 동시 노출
- 7일 테스트로 CTR, CPC, 전환율 데이터 확보 → 스케일업 판단 가능

**광고 설정 가이드:**
- **목표:** 트래픽 (웹사이트 방문)
- **타겟:** 18~35세, 한국, 관심사 "뉴스", "시사", "팩트체크"
- **소재:** 기존 OG 이미지 + "매일 아침 1분, AI가 팩트체크한 뉴스 브리핑" 카피
- **링크:** `factlens.pages.dev?utm_source=meta_ads&utm_medium=paid&utm_campaign=test_w1`
- **기간:** 7일 (테스트)
- **예산:** ₩5,000/일 (총 ₩35,000)

### 보조 채널: 네이버 검색광고 (검토용)

**이유:**
- "뉴스 요약 사이트", "팩트체크 사이트" 같은 키워드로 검색하는 사람 = 이미 니즈가 있는 사람
- 의도 기반이라 전환율이 디스플레이 광고보다 높음

**확인 필요:**
- 네이버 검색광고 키워드 도구에서 "뉴스 요약", "팩트체크" 월간 검색량 + 예상 CPC 확인
- CPC가 ₩500 이하면 시도 가치 있음

### 피드백 분석

**긍정 (3명: 감자, 가오나시, 애드센스)**
- 핵심 가치(팩트체크 태그) 호응: "진위여부 띄워주니 흥미롭고 보기 좋다"
- 요약 품질 긍정: "딱 필요한 내용만 요약해서 보여주니 이슈 파악이 쉽다"
- 가짜뉴스 검증 과정에 흥미: "어떻게 가짜로 보도가 됐는지 보는 게 흥미롭다"

**개선 요청 (2명: 애드센스, Kevin)**
- 가독성 이슈: "텍스트가 너무 많아 눈에 잘 안 들어온다"
- 핵심 단어 하이라이트 요청: "핵심 단어에 색칠되면 좋겠다"
- 반론: 감자/가오나시는 현재 상태 괜찮다고 평가 → 의견 분분

**결론**
- 서비스 핵심 가치(팩트체크 태그 + 쉬운 요약)는 검증됨
- 가독성은 개선 여지 있으나 의견이 갈림 → 점진적 개선 대상
- 핵심 단어 하이라이트는 "옵션"으로 제공하는 방향 검토 필요
