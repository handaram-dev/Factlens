## 1. Google AdSense

### 승인 조건
- **최소 트래픽**: 공식적으로 명시된 수치 없음. 다만 코드 삽입 후 "정기적으로 조회가 발생"해야 계정 활성화됨
- **콘텐츠 요건**: 고품질 독창적 콘텐츠, 최소 10~20개 게시글 권장, 게시글당 1,000자 이상 권장
- **기술 요건**: HTTPS 필수, sitemap.xml/robots.txt 권장, 만 19세 이상
- **승인 난이도**: 높음 (한국에서 승인 거절 사례 다수 보고됨)
- **승인 소요 시간**: 보통 며칠~2주, 경우에 따라 4주

### 수익 구조
- **과금 방식**: 2024년 2월부터 CPC에서 **CPM 중심**으로 전환
- **예상 단가**: 한국은 CPC 단가 국가 순위 56위로 낮은 편. 평균 CPC $0.1~$0.5 수준. 고단가 키워드(금융/보험/대출)는 $1 이상 가능
- **뉴스 카테고리**: 일반 뉴스는 단가가 낮은 편

### 정치/종교 광고 차단
- **가능**: 민감한 카테고리(Sensitive Categories)에서 "정치", "종교" 카테고리를 개별적으로 차단 가능
- 사이트별 차단 설정도 지원
- 단, 100% 차단을 보장하지는 않음 (Google 공식 안내)

### 정적 사이트 호환성
- **완전 호환**: JS 스니펫(`<script>` 태그)만 삽입하면 동작. 서버 사이드 로직 불필요
- Cloudflare Pages와 호환 문제 없음

### 최소 정산 금액
- **$100 USD** (약 13만 원)
- 해외 송금 수수료 발생 (중개은행 $5 + 시중은행 수수료 5,000~10,000원)
- SC제일은행은 $300 이하 수수료 면제

### 장점
- 글로벌 최대 광고 네트워크, 광고주 풀이 가장 넓음
- 단가가 국내 플랫폼 대비 상대적으로 높음
- 정치/종교 등 민감한 카테고리 세밀한 차단 가능
- 자동 광고 최적화(Auto Ads) 기능 제공

### 단점
- 승인 난이도 높음 (초기 소규모 사이트에서 거절 다수)
- 최소 정산 금액 $100로 높음 (소규모 사이트는 정산까지 수개월 소요)
- 한국 CPC 단가가 글로벌 대비 낮음
- 해외 송금 수수료 발생

### 초기 소규모 사이트 적합성: **보통**
승인만 받으면 최고의 선택이지만, 초기 단계에서 승인이 어렵고 $100 정산 기준액에 도달하기까지 오래 걸릴 수 있음.

## 2. 카카오 AdFit (애드핏)

### 승인 조건
- **최소 트래픽**: 공식 기준 미공개. 매체 심사 시 "트래픽의 질적/양적 품질" 확인
- **콘텐츠 요건**: 매체 운영 30일 이상, 포스팅 20개 이상 권장. 콘텐츠의 질적/양적 품질 심사
- **기술 요건**: AdFit SDK/스크립트 설치 후 광고 요청이 정상 동작해야 심사 진행 가능
- **승인 난이도**: 보통 (AdSense보다 낮음)
- **승인 소요 시간**: 영업일 1~2일 (상황에 따라 지연 가능)

### 수익 구조
- **과금 방식**: CPC + CPM 혼합 (실시간 입찰 방식)
- **예상 단가**: AdSense 대비 낮은 편. CPC 평균 약 20~50원 수준으로 알려져 있음
- **뉴스 카테고리**: 특화 데이터 없음

### 정치/종교 광고 차단
- **제한적**: 운영정책에서 광고 카테고리별 세밀한 차단 기능에 대한 공식 안내가 부족함
- AdSense처럼 민감한 카테고리를 세밀하게 차단하는 기능은 제공되지 않는 것으로 파악됨

### 정적 사이트 호환성
- **완전 호환**: JS 스크립트 삽입 방식. 정적 사이트에서 문제 없이 동작
- 웹(PC/모바일) 및 앱 모두 지원

### 최소 정산 금액
- **5만 원**
- 국내 계좌로 지급, 해외 송금 수수료 없음

### 장점
- 승인이 AdSense보다 쉬움 (영업일 1~2일)
- 국내 계좌 직접 지급, 해외 송금 수수료 없음
- 카카오 생태계(다음, 카카오톡) 기반의 광고 네트워크
- 한국어 고객 지원

### 단점
- CPC 단가가 AdSense 대비 낮음
- 정치/종교 광고 카테고리 차단 기능이 제한적
- 최소 정산 5만 원 (소규모 트래픽에서는 정산까지 시간 소요)
- 광고주 풀이 AdSense 대비 작음

### 초기 소규모 사이트 적합성: **높음**
승인이 비교적 쉽고, 국내 서비스라 정산이 편리. 단, 정치/종교 광고 차단 기능이 제한적인 점이 뉴스 사이트에는 우려 사항.

## 3. 쿠팡 파트너스

### 승인 조건
- **최소 트래픽**: 없음. 쿠팡 계정 + 본인 인증만으로 누구나 가입 가능
- **콘텐츠 요건**: 활동 채널(블로그, 웹사이트, SNS, 유튜브 등) URL 등록 필요
- **승인 난이도**: 매우 낮음 (사실상 즉시 가입)
- **승인 소요 시간**: 즉시~1일

### 수익 구조
- **과금 방식**: **CPS (Cost Per Sale)** — 판매 성사 시에만 수수료 지급
- **수수료율**: 기본 3% (카테고리별 상이)
- **쿠키 기간**: 링크 클릭 후 24시간 내 구매 시 수익 인정
- **뉴스 사이트 특성**: 뉴스 콘텐츠와 쇼핑 간 연관성이 낮아 전환율이 매우 낮을 수 있음

### 정치/종교 광고 차단
- **해당 없음**: 쿠팡 파트너스는 쿠팡 상품 링크/배너이므로 정치/종교 광고가 노출될 가능성 없음. 쿠팡 자체 광고 정책에서도 정치/종교 관련 콘텐츠 광고 금지

### 정적 사이트 호환성
- **완전 호환**: 상품 링크, 다이나믹 배너 등을 JS 스니펫으로 삽입 가능
- 단, 2025년 정책 변경으로 플로팅 배너, 더보기 배너 금지

### 최소 정산 금액
- **1만 원**
- 국내 계좌 지급, 수익 발생월 기준 +45일 (익익월 15일) 지급

### 장점
- 가입 즉시 사용 가능 (승인 절차 사실상 없음)
- 최소 정산 금액 1만 원으로 가장 낮음
- 정치/종교 광고 우려 없음 (쇼핑 상품만 노출)
- 국내 정산, 수수료 없음

### 단점
- CPS 방식이라 클릭만으로는 수익 발생 안 함 (구매 전환 필요)
- 뉴스 브리핑 서비스와 쇼핑의 연관성이 낮아 전환율 저조 예상
- 수수료율 3%로 낮음 (고가 상품이 아니면 건당 수익 미미)
- 24시간 쿠키 제한이 짧음

### 초기 소규모 사이트 적합성: **보통**
가입은 가장 쉽지만, 뉴스 브리핑 서비스 특성상 쇼핑 전환이 일어나기 어려워 실질 수익이 매우 낮을 가능성이 높음. 보조 수익원으로만 적합.

## 4. Ezoic

### 승인 조건
- **최소 트래픽**: 없음 (과거 월 10,000 페이지뷰 요건 폐지됨)
- **콘텐츠 요건**: 기본적인 콘텐츠 품질 요건 충족 필요
- **승인 난이도**: 낮음
- **승인 소요 시간**: 수일 이내

### 수익 구조
- **과금 방식**: CPM 중심 (AI 기반 광고 배치 최적화)
- **예상 단가**: EPMV(1,000 방문자당 수익) 기준 비영어권은 $5~$10 수준. 한국어 사이트는 낮은 편으로 예상
- **특징**: AI가 광고 배치를 자동 최적화하여 수익 극대화

### 정치/종교 광고 차단
- **가능**: Google Ad Exchange 기반이므로 AdSense와 유사한 카테고리 차단 기능 제공

### 정적 사이트 호환성
- **완전 호환**: JS 스니펫 방식. Cloudflare 통합 플러그인도 제공
- Cloudflare Pages와의 호환성 우수

### 최소 정산 금액
- **$20 USD** (약 2.6만 원) — AdSense의 1/5 수준
- PayPal, Payoneer, 은행 송금 등 다양한 지급 수단

### 장점
- 트래픽 제한 없이 가입 가능
- AI 기반 광고 최적화로 수동 관리 최소화
- 최소 정산 $20로 낮음
- Cloudflare 통합 지원
- AdSense보다 승인이 쉬움

### 단점
- 한국어 지원 미비 (영어 기반 서비스)
- 비영어권 사이트의 CPM 단가가 낮음
- 해외 송금 수수료 발생
- 한국 내 사용 사례/커뮤니티가 적어 정보 부족
- 사이트 속도에 영향을 줄 수 있음

### 초기 소규모 사이트 적합성: **보통**
트래픽 제한이 없어 가입은 쉽지만, 한국어 사이트의 CPM 단가가 낮고 영어 기반 서비스라 운영 부담이 있음.

## 5. 데이블 (Dable) — 네이티브 광고

### 승인 조건
- **최소 트래픽**: **일 페이지뷰 3,000 이상**, 콘텐츠 300개 이상
- **승인 난이도**: 높음 (소규모 사이트에는 진입장벽이 높음)
- **승인 소요 시간**: 매주 월/수/금 심사, 1주 이내

### 수익 구조
- **과금 방식**: CPC 기반
- **수익 배분**: 광고 수익의 50%를 매체에 지급 (사업소득 3.3% 원천징수)
- **정산 일정**: 익월 말일 정산

### 정치/종교 광고 차단
- **확인 필요**: 공식적으로 카테고리별 차단 기능에 대한 상세 안내 부재. 직접 문의 필요

### 정적 사이트 호환성
- **완전 호환**: JS 스크립트 삽입 방식으로 정적 사이트에서 동작

### 최소 정산 금액
- 별도 명시 없음 (익월 말일 정산)

### 장점
- 뉴스/콘텐츠 사이트에 최적화된 네이티브 광고
- "추천 콘텐츠" 형태로 사용자 경험을 덜 해침
- 한국 내 주요 뉴스 매체들이 사용하는 플랫폼

### 단점
- **일 PV 3,000 + 콘텐츠 300개** 요건이 초기 사이트에는 매우 높은 장벽
- 수익 배분 50%만 매체에 지급
- 최소 트래픽 미달 시 서비스 해지 가능

### 초기 소규모 사이트 적합성: **부적합**
일 PV 3,000 및 콘텐츠 300개 요건이 MVP 단계 사이트에는 현실적으로 불가능. 트래픽 성장 후 고려할 플랫폼.

## 6. 네이버 애드포스트

### 승인 조건
- **플랫폼 제한**: **네이버 블로그 전용** (외부 웹사이트 사용 불가)

## 7. 텐핑 (Tenping) — CPA 광고

### 승인 조건
- **최소 트래픽**: 없음 (누구나 가입 가능)
- **승인 난이도**: 매우 낮음

### 수익 구조
- **과금 방식**: **CPA (Cost Per Action)** — 클릭형, 참여형, 설치형, 구매형
- **수익 방식**: 사용자가 링크를 클릭하여 콘텐츠를 보거나 구매/참여/설치하면 캐시 적립
- **마케터 수**: 48만 명 이상 활동 중 (2025년 4월 기준)

### 정치/종교 광고 차단
- **해당 없음**: 매체 운영자가 노출할 캠페인을 직접 선택하는 구조이므로, 정치/종교 관련 캠페인을 선택하지 않으면 됨

### 정적 사이트 호환성
- **제한적**: 주로 블로그/SNS에 링크 삽입 방식으로 활용. 정적 사이트에 배너 형태로 삽입하기에는 구조적으로 부적합

### 최소 정산 금액
- **5,000원**

### 장점
- 가입 쉬움, 최소 정산 금액 낮음
- 캠페인을 직접 선택할 수 있어 광고 내용 통제 가능

### 단점
- 뉴스 브리핑 사이트와 CPA 광고 간 시너지가 낮음
- 정적 사이트에 자연스럽게 삽입하기 어려움
- 개별 캠페인 링크를 수동으로 관리해야 함
- 퍼포먼스 마케팅 플랫폼이라 디스플레이 광고와는 성격이 다름

### 초기 소규모 사이트 적합성: **낮음**
뉴스 브리핑 서비스의 UX와 맞지 않고, 수동 관리 부담이 큼.

## 8. 버즈빌 (Buzzvil) — 리워드 광고

### 승인 조건
- **대상**: 주로 앱 매체 및 대형 퍼블리셔 대상
- **승인 난이도**: 높음 (B2B 영업 문의 필요)

### 수익 구조
- **과금 방식**: CPA + CPM 혼합 (리워드 기반)
- **특징**: BuzzAd Feed(Web) 제품으로 웹에서도 사용 가능. 30분 내 연동 가능

### 정적 사이트 호환성
- **가능**: 웹 피드 형태로 삽입 가능하나, 주로 대형 매체 대상

### 초기 소규모 사이트 적합성: **부적합**
대형 퍼블리셔/앱 매체 위주의 B2B 서비스. 소규모 개인 사이트에는 적합하지 않음.

## 최종 추천 순위

팩트렌즈(AI 뉴스 브리핑, 정적 사이트, 초기 소규모, 정치/종교 광고 차단 필수) 기준:

### 1위: 카카오 AdFit + 쿠팡 파트너스 (병행)
- **이유**: 즉시 시작 가능한 현실적 조합
- AdFit: 승인 1~2일, 디스플레이 광고로 기본 수익 확보
- 쿠팡 파트너스: 즉시 가입, 뉴스와 연관된 상품 추천 배너를 사이드바에 배치하여 보조 수익
- **우려**: AdFit의 정치/종교 광고 차단 기능이 제한적. 뉴스 중립성 리스크가 존재하므로 노출되는 광고를 주기적으로 모니터링 필요

### 2위: Google AdSense
- **이유**: 정치/종교 카테고리 차단이 가장 세밀하고, 단가도 가장 높음
- 뉴스 콘텐츠와의 호환성이 가장 좋음
- **전략**: 콘텐츠 20개 이상 확보 후 승인 신청. 거절 시 콘텐츠 보강 후 재신청
- **우려**: 승인까지 시간이 걸리고, $100 정산 기준액 도달에 시간 소요

### 3위: Ezoic
- **이유**: AdSense 승인이 반복 거절될 경우의 대안
- 트래픽 제한 없이 가입 가능하고, Cloudflare 통합 지원
- AI 기반 광고 최적화로 수동 관리 부담 적음
- **우려**: 한국어 서비스 미비, 비영어권 사이트 단가 낮음

### 4위: 데이블 (Dable)
- **이유**: 뉴스/콘텐츠 사이트에 가장 최적화된 네이티브 광고 플랫폼
- **시기**: 일 PV 3,000 + 콘텐츠 300개 달성 후 신청
- 팩트렌즈가 성장 궤도에 오른 후(3~6개월 뒤) 고려할 플랫폼

## 권장 실행 계획

### Phase 1: MVP 즉시 적용 (지금)
1. **쿠팡 파트너스** 가입 (즉시) — 사이드바/하단에 관련 상품 배너 배치
2. **카카오 AdFit** 신청 (1~2일 소요) — 본문 상/하단에 디스플레이 배너 배치

### Phase 2: 콘텐츠 축적 후 (2~4주 뒤)
3. **Google AdSense** 승인 신청 — 콘텐츠 20개 이상 확보 후 신청
4. 승인 시: AdFit을 보조로, AdSense를 메인 광고로 전환
5. 거절 시: Ezoic 대안 검토

### Phase 3: 트래픽 성장 후 (3~6개월 뒤)
6. 일 PV 3,000 달성 시 **데이블** 신청 검토
7. AdSense + 데이블 병행으로 수익 극대화
