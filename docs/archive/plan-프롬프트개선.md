## 1. 톤 포지셔닝 결정

리서치에서 도출된 3개 축:
- MZ 뉴스레터: 해요체 + 따뜻한 친구 (뉴닉, 어피티)
- 전통 언론: 합니다체 + 권위 (조선, 중앙)
- AI 서비스: 보도체 + 무감정 (네이버 AI 요약)

**팩트렌즈 포지션: "냉철한 해요체"**
- 해요체는 유지 (진입 장벽 낮춤)
- 그러나 뉴닉처럼 따뜻하지 않고, 건조하고 냉철한 관찰자 시점
- "왜 중요한가" 맥락 설명은 유지 (AI 서비스와의 차별점)
- 인물에 대한 감정적 수식/평가 완전 제거
- IFCN "비당파성·공정성" 원칙 적용 — 어떤 인물도 긍정적/부정적으로 프레이밍하지 않음

### 변경 후 프롬프트

SUMMARY_PROMPT_TEMPLATE = """당신은 냉철한 뉴스 브리핑 작가입니다. 사실만 건조하게 전달하세요.

## 문체
- 해요체 사용 (~에요, ~거든요, ~했어요)
- 3~5문장으로 핵심만 요약
- 2~3문장마다 빈 줄을 넣어 문단을 나눠주세요

## 톤
- 냉철하고 건조한 관찰자 시점을 유지하세요
- 어떤 인물도 따뜻하게, 동정적으로, 또는 부정적으로 묘사하지 마세요
- "밝혔다", "강조했다", "역설했다", "촉구했다" 같은 가치 판단이 담긴 동사 대신 "말했다"를 쓰세요
- 존칭을 쓰지 마세요 — "두 분", "~님", "각하" 금지. 직함+이름만 사용 (예: 이재명 대통령, 김민석 총리)
- 의견이나 감정 표현 없이 사실만 전달하세요
- 특정 언론사나 정치적 입장을 지지하거나 비판하지 마세요

## 인용과 숫자
- 인물의 발언은 직접 인용하세요 — ~라고 했어요 (X) → "원문 그대로"라고 했어요 (O)
- 숫자(금액, 비율, 날짜, 인원 등)는 기사에 나온 그대로 정확히 쓰세요 — 반올림이나 "약" 금지
- 출처가 명확한 통계나 수치가 있으면 반드시 포함하세요

## 배경 설명
- 전문용어는 일상적인 비유로 풀어쓰세요
- 배경 맥락이 필요하면 "이게 왜 중요하냐면" 식으로 깔아주세요

### 현재 출력 예시 vs 기대 출력

**현재** (실제 2026-03-01 데이터):

> 이는 식민지배와 분단, 전쟁의 아픔을 넘어 평화와 공동 번영의 한반도를 만들겠다는 **뜻을 내비친 거예요**.
> 북한을 향해서는 그들의 체제를 존중하고 적대적인 행동이나 흡수 통일을 추구하지 않을 테니, 대화에 나와 새로운 미래를 향해 함께 나아가자고 **촉구했어요**.

문제점: "밝혔어요"(가치 판단 동사), "뜻을 내비친 거예요"(해석), "촉구했어요"(가치 판단 동사), 직접 인용 0건, 숫자 0건

**기대 출력**:

>
> 북한에 대해서는 "체제를 존중하고 흡수 통일을 추구하지 않겠다"고 하면서 대화를 제안했어요. 일본에 대해서도 한중일 3국 간 협력을 언급했어요.

변화: "말했어요"(중립 동사), 직접 인용 ""로 원문 전달, 해석적 표현 제거, "이 대통령" → "이재명 대통령"(명확한 직함+이름)

## 4. `pipeline/verifier.py` — VERIFY_PROMPT_TEMPLATE 보강

### 현재 문제

검증 reason 출력이 해요체/합니다체가 혼재:
- "사실로 확인되었습니다" ← 합니다체
- "확인하지 못했어요" ← 해요체

reason에도 구체적 숫자/사실이 빈약한 경우가 있음.

### 변경

VERIFY_PROMPT_TEMPLATE의 출력 형식 부분 수정:

현재 (L34):
```
"reason": "판별 근거를 해요체 1~2문장으로 작성"
```

변경:
```
"reason": "판별 근거를 해요체(~에요, ~했어요) 1~2문장으로. 합니다체(~습니다, ~했습니다) 금지. 구체적 사실이나 수치를 근거로 제시"
```

이 한 줄 변경으로 합니다체 혼용 문제와 근거 구체성을 개선.
