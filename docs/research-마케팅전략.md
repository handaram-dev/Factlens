# 리서치 — FactLens 초기 마케팅 전략

**Date:** 2026-03-02

---

## 1. 현황 분석

### 서비스 특성
- **URL:** factlens.pages.dev
- **한줄 소개:** AI가 뉴스를 쉬운 말로 요약하고, 팩트체크 검증 태그를 붙여주는 중립적 뉴스 브리핑
- **타겟:** 20~30대 뉴스 이탈층
- **핵심 가치:** ✅사실 / ⚠️논란 중 / ❌왜곡 태그 + 쉬운 요약 + 전체 언론사 원문 링크
- **비용:** 운영비 ₩0/월 (무료 티어)
- **업데이트:** 매일 오전 7시 자동 브리핑

### MVP 검증 목표 (4주)
| 지표 | 목표 |
|------|------|
| 7일 내 재방문율 | 30% 이상 |
| 일일 UV | 100 UV/일 |
| 공유 버튼 클릭 | 주 10건 이상 |

### 마케팅 예산
- **₩0** — 유료 광고 없이 오가닉 채널만 활용
- 1인 운영이므로 시간 효율이 최우선

## 2. 채널별 전략

### 2-1. Threads (쓰레드) — 핵심 채널

**왜 쓰레드인가?**
- 2026년 기준 알고리즘이 "대화의 깊이(Reply Depth)"를 핵심 노출 지표로 사용
- 텍스트 중심 플랫폼이라 뉴스/팩트체크 콘텐츠와 궁합이 좋음
- 팔로워 0명이어도 알고리즘에 의해 비팔로워에게 노출 가능
- 20~30대 사용자 비율이 높아 타겟과 일치

**콘텐츠 전략**

(1) **론칭 소개 글** — "빌딩 인 퍼블릭" 스타일
```
뉴스 읽기 싫어서 직접 만들었습니다.

매일 아침 AI가 뉴스를 쉬운 말로 요약해주고,
✅사실 / ⚠️논란 중 / ❌왜곡 태그까지 달아줍니다.

"조선일보랑 한겨레가 같은 사건을 다르게 쓰는데
둘 다 볼 시간은 없잖아요."
→ 그래서 모든 언론사 원문을 나란히 보여줍니다.

혼자 만든 사이드프로젝트인데, 한번 써보시고 피드백 주세요.
→ factlens.pages.dev

#팩트렌즈 #팩트체크 #뉴스요약 #사이드프로젝트
```

(2) **매일 뉴스 카드** — 반복 콘텐츠
- 매일 브리핑에서 가장 흥미로운 기사 1개를 뽑아 쓰레드에 공유
- 형식: "오늘 브리핑에서 뽑은 뉴스 한 줄" + 요약 + 검증 태그 + 링크 (운영자가 직접 선별)
- 텍스트 + OG 이미지 조합 (단독 텍스트보다 도달률 ~1.5배)

(3) **시리즈 콘텐츠** — "이번 주 팩트체크 리포트"
- 주 1회, 한 주간 검증 태그 통계 공유 (사실 몇 건, 논란 중 몇 건 등)
- "스레드 잇기" 기능으로 연속물 제작 → 팔로우 유도

**운영 팁**
- 질문형/공감형 오프닝으로 시작 ("이 뉴스 진짜일까?", "어제 이 뉴스 보셨나요?")
- 타 계정 쓰레드에 맥락 있는 댓글 적극적으로 달기 (수동 발행보다 효과적)
- 해시태그: `#팩트체크`, `#뉴스요약`, `#사이드프로젝트`, `#오늘의뉴스`

### 2-2. X (트위터) — 실사용자 확보 채널

**왜 X인가?**
- 실시간 트렌드와 뉴스 소비가 가장 활발한 플랫폼
- 뉴스/시사 관심층이 밀집 — 타겟과 직접 연결
- 리트윗으로 빠른 확산 가능

**콘텐츠 전략**

(1) **론칭 트윗** — 뉴스 소비 불편함 공감
```
뉴스 보고 싶은데 어렵고, 뭘 믿어야 할지 모르겠어서
직접 만들었습니다.

매일 아침 AI가 뉴스를 쉬운 말로 요약해주고,
✅사실 / ⚠️논란 중 / ❌왜곡 태그까지 달아줍니다.

전문용어는 "바우처(정부가 주는 쿠폰)" 이런 식으로 풀어주고,
같은 사건의 모든 언론사 원문도 비교할 수 있어요.

무료, 회원가입 없음, 아침 1분이면 끝.
→ factlens.pages.dev

#팩트체크 #뉴스요약 #오늘의뉴스 #팩트렌즈
```

(2) **매일 뉴스 카드** — 쓰레드와 동일하게 운영
- 매일 브리핑에서 한 건 선별하여 트윗
- 형식: 뉴스 한 줄 요약 + 검증 태그 + 링크

(3) **뉴스 반응형 트윗** — 실시간 트렌드 활용
- 핫한 뉴스가 나왔을 때 "팩트렌즈에서 이 뉴스 검증 결과" 트윗
- 트렌딩 해시태그와 연결

**운영 팁**
- 뉴스/시사 관심 계정들과 상호작용
- 해시태그: `#팩트체크`, `#뉴스요약`, `#오늘의뉴스`, `#팩트렌즈`
- 이미지 첨부 시 노출량 증가 (OG 카드 활용)

### 2-3. 에브리타임 — 대학생 직접 타겟

**왜 에브리타임인가?**
- MAU 300만, 누적 가입자 700만 — 20대 대학생 최대 커뮤니티
- 타겟(20대 뉴스 이탈층)과 정확히 일치
- 홍보게시판에서 무료 홍보 가능

**콘텐츠 전략**

(1) **공감형 소개 글** — 광고 느낌 최소화
```
[제목] 뉴스 안 보는 사람 손🙋

솔직히 뉴스 어렵지 않나요?
전문용어에 숫자 범벅이고, 어떤 언론사를 믿어야 할지도 모르겠고...

그래서 만들었습니다.
AI가 매일 아침 뉴스를 쉬운 말로 요약해주고,
✅사실 ⚠️논란 중 ❌왜곡 태그까지 달아주는 서비스예요.

"바우처(정부가 주는 쿠폰)" 이런 식으로
전문용어도 괄호로 풀어서 설명해줍니다.

무료이고, 회원가입도 없어요.
아침에 1분이면 오늘 뉴스 끝!

👉 factlens.pages.dev

혼자 만든 사이드프로젝트인데 피드백 주시면 감사하겠습니다 🙏 
```

(2) **자유게시판 활용** — 뉴스 토론 유도
- "오늘 이 뉴스 진짜야?" 같은 질문형 글 → 자연스러운 서비스 언급
- 단, 과도한 홍보는 역효과 — 자연스러운 맥락에서만

**운영 팁**
- **게시 시간:** 월~목, 오전 10시~오후 11시 (주말 피하기)
- **톤:** 대학생 눈높이, 친근한 말투 (존댓말 OK, 광고체 NO)
- **핵심:** "무료 + 회원가입 없음 + 1분" 강조 — 진입 장벽 제로 어필
- **주의:** 에브리타임은 대학생 인증 필요 — 본인이 대학생이어야 직접 홍보 가능. 아니라면 지인 부탁 또는 대행사 활용 필요

### 2-4. 디스콰이엇 — 메이커 커뮤니티

**왜 디스콰이엇인가?**
- 한국판 Product Hunt — 누적 사용자 23만, 월 600개+ 프로덕트 포스트
- 사이드프로젝트에 관심 있는 메이커/개발자가 주 사용자
- 투표 + 피드백 시스템으로 초기 사용자 + 질적 피드백 동시 확보

**콘텐츠 전략**

(1) **프로덕트 등록**
- 서비스 소개, 스크린샷, 기술 스택 등 상세히 기재
- "문제 → 솔루션 → 차별점" 구조로 소개

(2) **메이커 로그** — 개발 과정 공유
- "₩0으로 뉴스 서비스 런칭한 이야기" 같은 빌딩 로그
- 기술적 선택(Gemini, GitHub Actions 등) 과정 공유

**기대 효과**
- 얼리어답터 확보 + 유의미한 피드백
- 개발자 네트워크 확장 → 2차 공유 효과

**콘텐츠 전략**

(1) **Show GN 포스트**
- 제목: "Show GN: 팩트렌즈 — AI 팩트체크 뉴스 브리핑 (₩0/월)"

## 6. 벤치마크 — 뉴닉의 초기 성장 전략

뉴닉(NEWNEEK)은 팩트렌즈와 유사한 "쉬운 뉴스" 서비스로, 2개월 만에 구독자 1만 명을 달성한 사례:

| 전략 | 내용 | 팩트렌즈 적용 |
|------|------|--------------|
| **컨닝페이퍼** | SNS에서 뉴닉을 쉽게 추천할 수 있는 공유 장치 | OG 태그 + 공유 버튼 (이미 구현) |
| **콘텐츠 접근성** | "망 = 고속도로 통행요금 같은 거예요" 수준의 쉬운 풀이 | 괄호 설명 의무화 (이미 구현) |
| **자발적 공유** | 유용한 콘텐츠 → 사용자가 알아서 공유 | 매일 SNS 카드 포스팅으로 공유 촉진 |

**팩트렌즈만의 차별점:**
- 뉴닉에 없는 **팩트체크 검증 태그** → 독보적 가치
- 뉴닉에 없는 **전체 언론사 원문 비교** → 편향 해소
- **₩0 운영 + 오픈소스** → 개발자 커뮤니티 어필
